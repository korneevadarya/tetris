# tetris

Классическая реализация игры «Тетрис» на Python с использованием графического фреймворка [Kivy](https://kivy.org/).

## 📖 Описание
Лёгкий и полностью рабочий клон тетриса, написанный на чистом Python. Игра включает все базовые механики: движение и вращение фигур, ускоренное падение, подсчёт очков, предпросмотр следующей фигуры, а также обнаружение завершения игры и возможность перезапуска без перезапуска приложения.

## ✨ Возможности
- 🧩 7 стандартных фигур тетриса
- 🔍 Предпросмотр следующей фигуры на боковой панели
- 📊 Система очков 
- 🔄 Автоматическое удаление заполненных горизонтальных рядов
- 💀 Экран Game Over с функцией новой игры
- 📏 Адаптивный размер игрового поля под ширину окна
